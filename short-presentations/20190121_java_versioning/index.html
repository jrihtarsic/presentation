Release requirements

Enterprise users

  1. Upgrading jdk usually cause costs

  2. Stability
  3. Long term support for the release
  4. 2-3y release is ok

Developers / software vendors

  1. New features and new technologies / standards reduces development costs

  2. New features as quick as possible
  3. Releases on time
  4. 2-3y for next major feature release is too long
